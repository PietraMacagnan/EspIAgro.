# EspIAgro Frontend

Frontend oficial do **EspIAgro**, plataforma de monitoramento inteligente de lavouras.

Esta aplicação será responsável por consumir o backend Django/DRF do projeto e oferecer a interface do usuário para:

- autenticação
- dashboard operacional
- monitoramentos em campo
- alertas automáticos
- relatórios técnicos
- base técnica de conhecimento
- evolução futura para mapas, clima e IA aplicada

---

## Stack atual

- **React**
- **Vite**
- **TypeScript**
- **React Router**
- **TanStack Query**
- **Axios**
- **MUI**
- **ESLint**

---

## Objetivo desta fase

Nesta fase, o frontend está sendo estruturado de forma **profissional, limpa e escalável**, tomando como base visual o protótipo aprovado do projeto e integrando com o backend já desenvolvido.

O foco atual é montar a base para:

- layout responsivo mobile-first
- navegação principal
- integração com API
- autenticação JWT
- consumo de monitoramentos, alertas e relatórios
